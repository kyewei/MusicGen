This browser does not support Applets.
Note:

This applet has the ability to write to file.

On newer versions of Chrome, NPAPI support is disabled, which means running Java applets is disabled unless a flag in chrome://flags/ called "Enable NPAPI" is ticked. Unfortunately, this flag may not be available in future versions of Chrome :(

If you are running Oracle Java, you might be able to run the applet without allowing full system access, but you won't be able to export files unless you agree to the security prompt. On newer versions of Oracle Java (8), you may need to add the certificate to "Trusted Certificates" and "Signed CA" in settings.

If you are running Iced-Tea Web, this applet/jar is self-signed so that on some versions it can be run once you accept the security prompt.
